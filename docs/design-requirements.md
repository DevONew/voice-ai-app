# 음성 AI 웹앱 디자인 요구사항

## 현재 작업 범위
**디자인 및 UI/UX만 집중** - 백엔드 API 연동은 내일 진행
- Web Speech API → Mock (버튼 클릭 시 더미 텍스트)
- Claude API → Mock (더미 응답)
- ElevenLabs API → Mock (애니메이션만)

## 기술 스택
- Next.js 14 + TypeScript + Tailwind CSS
- App Router 사용

---

## 화면 레이아웃 및 디자인

### 전체 레이아웃
- **모바일 브라우저 전용** (모바일 뷰포트 크기에 최적화)
- 전체 화면(100vh) 사용
- 배경: 깔끔한 단색 (흰색 또는 밝은 회색) 또는 은은한 그라데이션
- 여백: 적절한 패딩으로 답답하지 않게

### 중앙 마이크 버튼 (핵심!)
- **위치**: 화면 정중앙 (수직, 수평 모두 센터)
- **기본 크기**: 200px × 200px (또는 화면 너비의 40-50%)
- **색상**: 검정색 (#000000)
- **모양**: 완벽한 원형 (border-radius: 50%)
- **아이콘**: 
  - 흰색 마이크 아이콘 중앙 배치
  - 적절한 크기 (40-50px)
  - SVG 또는 아이콘 라이브러리 사용
- **그림자**: 은은한 shadow로 입체감 (optional)
- **호버/터치 효과**: 
  - 클릭 시 약간 scale down (0.95)
  - 부드러운 transition

### 원 애니메이션 (Mock)
- **평소**: 고정된 크기
- **"말하는 중" Mock 상태**: 
  - 0.5초마다 랜덤하게 크기 변화
  - 최소: scale(0.9) - 기본 크기의 90%
  - 최대: scale(1.2) - 기본 크기의 120%
  - 부드러운 ease-in-out transition
  - (실제 음성 볼륨은 내일 연동)

### 상태 표시 텍스트
- **위치**: 마이크 버튼 상단, 적당한 간격 (60-80px 위)
- **기본 상태**: "탭하여 시작" 
  - 색상: 중간 회색 (#666)
  - 크기: 16-18px
- **듣는 중 상태**: "듣는 중..."
  - 색상: 진한 회색 또는 검정
  - 약간 커진 크기 (18-20px)
  - fade-in 애니메이션
- **음성 인식 텍스트 (Mock)**: 
  - 버튼 클릭 후 2초 뒤 "안녕하세요, 테스트 음성입니다" 같은 더미 텍스트 표시
  - 폰트: 가독성 좋은 산세리프체 (예: Pretendard, Inter)
  - 크기: 18-20px
  - 색상: 검정 (#000) 또는 어두운 회색
  - 정렬: 중앙 정렬
  - 여러 줄 지원 (말 길어질 경우 대비)
  - 최대 너비: 화면의 80-90%

### 하단 상태 인디케이터 (Optional)
- 작은 점이나 텍스트로 현재 상태 표시
- 예: "대기 중" / "듣는 중" / "응답 중"
- 위치: 화면 하단 중앙
- 크기: 작게 (12-14px)
- 색상: 연한 회색

## 인터랙션 플로우 (Mock)

### 1. 초기 상태
- 검정색 원형 버튼 (정지)
- 위에 "탭하여 시작" 텍스트

### 2. 버튼 클릭 → Mock 녹음 시작
- "탭하여 시작" → "듣는 중..." 텍스트 변경 (fade 효과)
- 버튼에 미묘한 펄스 효과 또는 외곽선 애니메이션
- 2초 후 더미 텍스트 표시: "안녕하세요, 테스트 음성입니다"

### 3. Mock 음성 애니메이션
- 원이 자동으로 랜덤하게 커졌다 작아졌다 (0.5초 간격)
- 3-4초 정도 애니메이션 후 멈춤

### 4. 다시 클릭 → 초기화
- 텍스트 사라짐
- 다시 "탭하여 시작" 상태로

## 반응형 디자인

### 모바일 최적화
- **세로 모드 우선**
- 최소 너비: 320px (iPhone SE)
- 최대 너비: 430px (큰 스마트폰)
- 가로 모드는 고려하지 않음 (세로 전용)

### 터치 최적화
- 버튼 터치 영역 충분히 크게 (최소 44×44px)
- 터치 피드백 명확하게
- 스크롤 방지 (필요시)

---

## 디자인 스타일

### 컬러 팔레트
- 주 색상: 검정 (#000000) - 버튼
- 배경: 흰색 (#FFFFFF) 또는 밝은 회색 (#F5F5F5)
- 텍스트: 검정 (#000) ~ 중간 회색 (#666)
- 액센트: 필요시 포인트 컬러 추가 가능

### 타이포그래피
- 폰트: 깔끔한 산세리프 (Pretendard, Inter, SF Pro 등)
- 행간: 1.5-1.6
- 여백: 넉넉하게

### 애니메이션 원칙
- 모든 transition: 0.3-0.5초
- easing: ease-in-out 또는 cubic-bezier
- 과하지 않게, 부드럽게
- 60fps 유지

## 기술적 구현 가이드

### Mock 구현
```typescript
// 버튼 클릭 시
const handleClick = () => {
  // 1. 상태 변경: "듣는 중..."
  // 2. 2초 후 더미 텍스트 표시
  setTimeout(() => {
    setText("안녕하세요, 테스트 음성입니다");
  }, 2000);
  
  // 3. 원 애니메이션 시작 (랜덤 scale)
  // 4. 5초 후 애니메이션 정지
}
```

### 애니메이션
- CSS transitions 또는 Framer Motion 사용
- requestAnimationFrame 활용 (부드러운 애니메이션)
